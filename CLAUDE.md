# Studio Toorn - Premium Digital Agency Website

## Doel
Portfolio en agency website voor Studio Toorn, een full-service digital agency gevestigd aan de Costa del Sol, Spanje. Dit is de hoofdsite waarop we klanten binnenhalen.

## Tech Stack
- HTML + Tailwind CSS via CDN
- GSAP + ScrollTrigger + Lenis smooth scroll via CDN
- Statisch, één pagina
- Meertalig: Engels als default, met Spaans en Nederlands als opties

## Design Richting
Dark luxury tech aesthetic. Denk aan het niveau van agencies zoals Resn.co.nz, Active Theory, of Locomotive.ca. GEEN blauw/paars. Kleurenpalet: zwart/near-black basis, warm goud/amber accenten, crème voor tekst.
